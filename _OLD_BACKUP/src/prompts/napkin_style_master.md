## Napkin-Sketch Stil-Definition (FINAL)

### Visuelle Elemente

| Element | Spezifikation |
|---------|---------------|
| **Hintergrund** | Weiß/Serviette, subtile Textur, KEINE Flecken |
| **Linien** | Schwarz, wie mit Filzstift gezeichnet |
| **Farbakzent** | NUR Sage Green `#78866b` für Schraffur |
| **Handschrift** | Imperfekt, leicht wackelig, aber lesbar |
| **Icons** | Erlaubt: Sonne, Gras, Pfeile, einfache Symbole |
| **Achsen** | Klare schwarze Pfeile bei Diagrammen |

### Regeln

1. **Eine Grafik = Eine Message**
2. **Wenig Text:** Max. 5-7 Wörter
3. **Sprache:** Deutsche Beschriftung
4. **Farben:** NUR Schwarz + Sage Green, keine anderen Farben
5. **Rechtschreibung:** Exakt, keine Abkürzungen

### Wo Napkin-Sketches einsetzen

| Content-Typ | Napkin? |
|-------------|---------|

## Napkin-Ideen nach Thema

| Thema | Napkin-Sketch Idee |
|-------|-------------------|
| **Zeitverläufe** | Kurve mit Zeitachse (wie Fruktan) |
| **Vergleiche** | Zwei Seiten gegenüber (vs.) |
| **Prozesse** | Pfeile zwischen Schritten |
| **Körper-Abläufe** | Vereinfachte Anatomie-Skizze |
| **Checklisten** | Handgezeichnete Checkboxen |
| **Warnung/Gefahr** | Ausrufezeichen mit kurzem Text |
| **Temperatur/Risiko** | 2x2 Matrix mit Icons |
| **Ursache-Wirkung** | A → B Pfeil-Diagramm |

## Modul-Analyse: Visuelle Prüfung

Bei der Analyse jedes Modul-Abschnitts diese Kategorien prüfen:

```markdown
**Visuell:**
- [NAPKIN] Vorschlag: ... (Prozess/Zusammenhang der von Skizze profitiert)
- [ORGANIC] Vorschlag: ... (Emotionales Bild, Stimmung)
- [KEIN BILD] Hier reicht Text
```

### Entscheidungshilfe

| Frage | Ja → | Nein → |
|-------|------|--------|
| Erklärt einen Prozess/Ablauf? | NAPKIN | weiter |
| Zeigt einen Vergleich/Gegensatz? | NAPKIN | weiter |
| Ist ein Zeitverlauf/Kurve? | NAPKIN | weiter |
| Soll Emotion/Stimmung erzeugen? | ORGANIC | weiter |
| Ist Marketing/Cover/Header? | ORGANIC | weiter |
| Reine Information ohne visuelle Hilfe? | KEIN BILD | - |

## Technische Umsetzung

### Im Bulk Generator

```csv

fruktan_01,"[Inhalt]",Text1|Text2|Text3,output.png,napkin_pencil,"Beschreibung"
```

### Qualitätsprüfung

Erwartete Texte für OCR-Check:
- Alle Beschriftungen als `expected_texts` in CSV angeben
- Mindest-Ähnlichkeit: 80%
- Bei Rechtschreibfehlern: Auto-Retry (max. 3x)
